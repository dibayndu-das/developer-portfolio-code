What I Do ?

React

React, a JavaScript library, simplifies building interactive user interfaces. Its component-based architecture, virtual DOM, and strong community support make it a powerful tool for modern web development.

JavaScript

JavaScript is a versatile programming language for web development. It empowers interactive web pages, enhances user experiences, and supports both front-end and back-end development, making it essential in today's tech landscape.

Mongo DB

MongoDB is a NoSQL database known for its flexibility and scalability. It stores data in JSON-like documents, enabling efficient handling of large volumes of unstructured or semi-structured data in modern applications.

Node.js

Node.js is a runtime environment for executing JavaScript code outside of a web browser. It's widely used for server-side programming, offering high performance and non-blocking I/O for building scalable applications.
